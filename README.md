# is-valid-oib

Check if [OIB](http://en.wikipedia.org/wiki/Personal_identification_number_(Croatia)) is valid.

## Installation

```sh
npm install is-valid-oib --save
```

## Examples

```js
var valid = require('is-valid-oib');

valid('40565203589'); // => true
valid('29704949120'); // => false
```
